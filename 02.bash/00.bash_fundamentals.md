# Bash Scripting

## 1. What is Bash?

**Definition:**  
Bash (Bourne Again SHell) is a Unix shell and command language used for executing commands, scripting, and automating tasks in Linux and macOS environments.

**Example:**  
Open a terminal and type:
```bash
echo "Hello, Bash!"
```

---

## 2. Running Commands

**Definition:**  
You can run commands directly in the Bash shell or via scripts.

**Example:**  
```bash
ls -l
pwd
```

---

## 3. Scripts

**Definition:**  
A Bash script is a file containing a series of Bash commands.

**Example:**  
Create a file `myscript.sh`:
```bash
#!/bin/bash
echo "Script executed!"
```
Run it:
```bash
chmod +x myscript.sh
./myscript.sh
```

---

## 4. Variables

**Definition:**  
Variables store data for use in scripts.

**Example:**  
```bash
name="Nitin"
echo $name
```
⚠️ No spaces around `=`.

---

## 5. Environment Variables & Path

**Definition:**  
Environment variables are global variables available to all processes.  
`PATH` is an environment variable specifying directories for executable search.

**Example:**  
```bash
export PROJECT="DevOps"
echo $PROJECT
echo $PATH
```

---

## 6. Special Variables

**Definition:**  
Bash provides special variables for script arguments and status.

| Variable | Meaning                  |
|----------|-------------------------|
| $0       | Script name             |
| $1, $2   | First, second argument  |
| $#       | Number of arguments     |
| $@       | All arguments           |
| $?       | Exit status of last cmd |

**Example:**  
```bash
echo "Script name: $0"
echo "First argument: $1"
echo "Total arguments: $#"
```

---

## 7. Conditional Operators

**Definition:**  
Used to compare values in Bash.

| Operator | Meaning      |
|----------|-------------|
| -eq      | Equal       |
| -ne      | Not equal   |
| -gt      | Greater     |
| -lt      | Less        |
| -ge      | Greater/Equal|
| -le      | Less/Equal  |

**Example:**  
```bash
a=5
b=10
if [ $a -lt $b ]; then
    echo "$a is less than $b"
fi
```

---

## 8. If Statement

**Definition:**  
Conditional execution based on test results.

**Example:**  
```bash
num=10
if [ $num -gt 5 ]; then
    echo "Greater"
else
    echo "Not greater"
fi
```

---

## 9. Case Statement

**Definition:**  
Multi-way branch statement for matching patterns.

**Example:**  
```bash
read -p "Enter choice: " choice
case $choice in
    1) echo "One";;
    2) echo "Two";;
    *) echo "Other";;
esac
```

---

## 10. Pipelines & Logical Operators

**Definition:**  
- **Pipeline (`|`)**: Passes output of one command as input to another.
- **Logical Operators**:  
  - `&&`: Run next command if previous succeeds  
  - `||`: Run next command if previous fails

**Example:**  
```bash
cat file.txt | grep "ERROR"
ls && echo "Listed successfully"
ls invalidfile || echo "Failed to list"
```

---

## 11. For Loop

**Definition:**  
Iterates over a list or range.

**Example:**  
```bash
for i in {1..3}
do
    echo $i
done
```

---

## 12. While Loop

**Definition:**  
Repeats commands while a condition is true.

**Example:**  
```bash
count=1
while [ $count -le 3 ]
do
    echo $count
    ((count++))
done
```

---

## 13. Until Loop

**Definition:**  
Repeats commands until a condition becomes true.

**Example:**  
```bash
count=1
until [ $count -gt 3 ]
do
    echo $count
    ((count++))
done
```

---

## 14. Positional Arguments

**Definition:**  
Arguments passed to a script, accessed as `$1`, `$2`, etc.

**Example:**  
Script `args.sh`:
```bash
#!/bin/bash
echo "First: $1"
echo "Second: $2"
```
Run:
```bash
./args.sh apple banana
```

---

## 15. Input/Output

**Definition:**  
Redirect input/output streams.

**Example:**  
```bash
echo "Hello" > output.txt      # Output to file
cat < output.txt              # Input from file
ls invalidfile 2> error.log   # Error to file
```

---

## 16. Functions

**Definition:**  
Reusable code blocks in scripts.

**Example:**  
```bash
greet() {
    echo "Hello $1"
}
greet Nitin
```
