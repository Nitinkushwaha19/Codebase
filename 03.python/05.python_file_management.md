## Directory Listing

### Definition
`os.scandir()` is a function from the Python **os module** that is used to **iterate through files and directories inside a folder**.  
It returns directory entries that provide information like file name, file type, etc.

---

### List only files

```python
with os.scandir('.') as entries:
    for entry in entries:
        if entry.is_file():
            print(entry.name)
```

**Output**

```
app.py
data.txt
```
---

* `os.scandir()` → scans directory contents
* `entry.name` → returns file or folder name
* `entry.is_file()` → checks if entry is a file
* `entry.is_dir()` → checks if entry is a directory

## Making Directories in Python

### Create a Single Directory

```python
import os

os.mkdir('example_directory')
````

**Explanation**

* `os.mkdir()` creates **one new folder** in the current directory.
* It will give an **error if the folder already exists**.

**Example Structure**

```
.
└── example_directory/
```

---

| Function        | Purpose                            |
| --------------- | ---------------------------------- |
| `os.mkdir()`    | Create a single directory          |
| `os.makedirs()` | Create multiple nested directories |
